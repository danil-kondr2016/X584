Панель для точек останова и курсора выполнения микропрограммы

Панель может находиться в четырех состояниях:

  1. Исходное состояние, при котором происходит стандартная проверка функциональности кода микропрограммы.
  2. Состояние, характеризующее точку останова, т.е. в данном месте программа при выполнении микропрограммы должна остановиться.
  3. Состояние, показывающее, что на данном этапе происходит выполнение микропрограммы.
  4. Состояние, показывающее, что выполнение микропрограммы достигло точки останова, и ждет дальнейших действий.
