О программе

X584 — это программа, представляющая собой эмулятор ЦПЭ (центрального процессорного элемента) К584ВМ1.

Цель этой программы — развитие навыков работы на имитационных моделях отладочных средств при проектировании микропрограммируемых вычислительных устройств на базе микропроцессора серии К584.

В этой программе разработан удобный для пользователя интерфейс, условный переход (функционально в ЦПЭ он отсутствует). Добавлен он для более быстрой проверки работы созданного проекта. Создана гибкая отладка проекта, визуализация системы микрокоманд ЦПЭ К584ВМ1.

Программа имитирует работу четырёх ЦПЭ К584ВМ1, соединённых для наращивания разрядности. Разрядность блока составляет 16 бит, что соответствует десятичному числу 65535.
